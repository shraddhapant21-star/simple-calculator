## Calculator App

A simple Calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean interface and input validation.

## Features

Basic operations: + , − , × , ÷
Clear all (AC) and delete last digit (DE)
Decimal support
Prevents invalid expressions
User-friendly error handling

## Built With

HTML, CSS, JavaScript

## What I Learned

Handling user input with JavaScript
Performing calculations dynamically
Preventing invalid expressions using regex
Basic error handling with try...catch

## How to Use

Click the number and operator buttons
Press = to calculate
Use AC to clear or DE to delete the last input
